# Habito – Wellness & Habit Tracking Android Application

## Overview
Habito is a wellness-focused Android application developed using Kotlin and Android Studio to help users build healthy habits and maintain emotional well-being.

## Features
- Daily Habit Tracker
- Mood Journal with Emoji Selector
- Hydration Reminder
- Mood Analytics
- Progress Tracking
- Sensor-Based Mood Logging
- SharedPreferences Data Storage

## Technologies Used
- Kotlin
- Android Studio
- Android SDK
- SharedPreferences
- Material Design

## Skills
Android Development, Mobile Application Development, Kotlin, UI/UX Design, Sensor Integration
